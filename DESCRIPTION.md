# Описание проекта и инструкция по использованию

## 📖 О проекте

Этот проект представляет собой **одностраничный сайт‑приглашение** на свадьбу, реализованный с использованием чистого **HTML**, **CSS** и **Vanilla JavaScript**. Сайт включает:

- Эффект плавного появления элементов при скролле (Intersection Observer).
- Анимации и плавные переходы, выполненные с помощью CSS‑переменных и keyframes.
- Таймер обратного отсчёта до даты свадьбы.
- Раздел «Наша история» (таймлайн), секцию с деталями торжества, фотогалерею, цитаты о любви.
- **Форму‑анкету (RSVP)**, состоящую из пяти блоков (личные данные, присутствие, предпочтения в еде, трансфер/размещение, пожелания). Данные по умолчанию отправляются в консоль; при желании их можно отправлять в Formspree, Google Forms, EmailJS или собственный бекенд.
- Адаптивный дизайн: полностью работает на мобильных устройствах, планшетах и десктопе.
- Возможность легко менять цвета, шрифты и изображения через переменные CSS и несколько строк кода.

## 🚀 Как запустить сайт локально

1. **Открыть файл** `index.html` двойным кликом в браузере — быстрое предпросмотр.
2. Для более корректного поведения (особенно при работе с формой) рекомендуется запустить локальный сервер:
   ```bash
   # Если установлен Node.js
   npx serve .   # или npx http-server .

   # Если установлен Python
   python -m http.server 8080
   ```
   Затем открыть `http://localhost:8080` (или соответствующий порт) в браузере.

## 🔧 Как настроить сайт под свою свадьбу

### 1. Тексты и имена
- Отредактируйте **Hero‑секцию** (строки 36‑39 в `index.html`): замените `Александр`, `Мария` и дату.
- Обновите `<title>` и `<meta name="description">` в `<head>`.
- В разделах «Детали», «Программа», «Дресс‑код», «Контакты» замените тексты, даты, места и ссылки.

### 2. Таймер
- В `script.js` найдите строку:
  ```js
  const weddingDate = new Date('2026-06-14T14:00:00');
  ```
  Замените на дату и время вашего мероприятия.

### 3. Фотографии
- Поместите ваши фотографии в папку `photos/` рядом с `index.html`.
- В `style.css` замените свойства `background-image` классов `.gi-1 … .gi-6` на пути к вашим файлам.

### 4. Цитаты о любви (карусель)
- Добавьте новые карточки в блок `#wishes` в `index.html` и соответствующие точки‑индикаторы в `#carouselDots`.

### 5. Анкета (RSVP)
- По умолчанию данные выводятся в консоль. Чтобы сохранять ответы:
  - **Formspree**: зарегистрируйтесь, получите endpoint, замените URL в `script.js` (строка `fetch('https://formspree.io/f/…')`).
  - **EmailJS**: подключите библиотеку, укажите `service_id`, `template_id`, `user_id`.
  - **Google Forms / Airtable**: вставьте iframe вместо формы.
  - **Свой сервер**: укажите ваш URL в `fetch()`.
- При желании убрать/добавить блоки анкеты просто удалите/скопируйте соответствующий `<fieldset class="f-block …">` из `index.html`.

### 6. Цветовая схема
- В `style.css` в `:root` находятся CSS‑переменные:
  ```css
  :root {
    --primary: hsl(340, 70%, 50%);   /* основной цвет */
    --secondary: hsl(210, 15%, 95%); /* фон */
    --accent: hsl(45, 90%, 55%);    /* акцент */
    ...
  }
  ```
  Измените их под ваш стиль (пастельные тона, золотой, морской и т.д.).

### 7. Музыка (опционально)
- Добавьте в `index.html` элемент:
  ```html
  <audio id="bg-music" src="music.mp3" loop></audio>
  ```
- Добавьте кнопку включения/выключения и логику в `script.js`.
